En Angular, los "pipes" (también conocidos como "tuberías" o "filtros") son una característica que permite transformar datos en una plantilla de manera declarativa. Los pipes toman una entrada de datos, la procesan y devuelven una salida que se puede mostrar en la plantilla, los datos no se modifican directamente solo se muestran con un formato especifico, pero se mantiene su estado normal

Con PIPES: {{'hello world' | uppercase }} => (uppercase de PIPES hace mayuscula una cadena de caracteres)
Sin PIPES: {{'hello world'}}

Con PIPES: {{ 'HELLO WORLD' | lowercase }} => (lowercase de PIPES hace mayuscula una cadena de caracteres)
Sin PIPES: {{'HELLO WORLD'}}

Con PIPES: {{ date | date:'fullDate' }} => (date: 'fullDate' genera un formato especifico a la fecha)
Sin PIPES: {{date}}

Con PIPES: {{ 1000 | currency:'USD':true }} => (currency:'USD':true genera un formato de moneda al valor numerico)
Sin PIPES: {{1000}}

Con PIPES: {{ 3.14159 | number:'1.2-2' }} => (3.14159 | number:'1.2-2' genera un formato de numeros decimales)
Sin PIPES: {{3.14159}}

Con PIPES: {{ 0.25 | percent }} => (0.25 | percent genera un formato de porcentaje al valor entante)
Sin PIPES: {{0.25}}

Con PIPES: {{ user | json }} => (user | json genera un JSON con los parametros como cadena de caracteres u objetos)
Sin PIPES: {{user}}
